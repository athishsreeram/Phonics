# 📝 Phonics Learning Hub

Welcome to the **Phonics Learning Hub** — an interactive, web-based platform designed to help young learners (ages 3–7) practice phonics and early reading skills.

This project is built with **HTML, CSS, and JavaScript** for **personal and classroom learning only**.

---

## 🌐 Live Demo

Try it in your browser: [Phonics Learning Hub](https://athishsreeram.github.io/Phonics/index.html)

---

## 🎯 Features

- Interactive phonics lessons for beginners  
- Simple, child-friendly UI  
- Works offline in the browser  
- Designed for **personal or classroom learning**  

---

## 📂 Project Structure

```

Phonics/
│
├─ index.html           # Main landing page
├─ style.css            # Styling for pages
├─ script.js            # Interactive behavior
├─ assets/              # Images, sounds, icons
└─ lessons/             # Individual phonics lessons (HTML/JS)

````

---

## 🚀 How to Use

1. Clone the repo:

```bash
git clone https://github.com/athishsreeram/Phonics.git
````

2. Open `index.html` in your browser.
3. Explore the phonics lessons and practice reading!
